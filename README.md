# fortran wrapper to vesin

A slightly more sophisticated fortran wrapper to `vesin` from: https://github.com/Luthaf/vesin

# compile

```bash
git clone git@github.com:mgoonde/f_vesin.git
git submodule update --init
make
```

clean
```bash
make clean
```

# run

```bash
cd test
./main.x < xx.xyz
```
